私有权限

定义的函数和成员类型以__开头就可以了。这样，类外无法访问，并且其他类也无法继承到私有的成员和函数。想要调用或者修改，必须在类内重新写函数

多态

多态在我的C++中的理解，就是写一个虚函数的框架，其他的子类在这个框架的基础上进行修改，其实python学习上面的继承中已经体现到了多态，就是基于函数的重写，并且多态的优势在于主框架的指针能够指向其他的重写子类

class Dog():
    def work(self):
        print('xuhanshu')
class armydog(Dog):
    def work(self):
        print('armydog fight')
class drugdog(Dog):
    def work(self):
        print('drug dog fight')
class person():
    def person_with_dog(self,ddog):
        ddog.work()
a=armydog()
b=drugdog()
p=person()
p.person_with_dog(a)
p.person_with_dog(b)

静态方法

class Dog():
    __name='ddd'
    def __init__(self):
        age=1
    @staticmethod
    def work():#没有参数传递
        print('xuhanshu')

a=Dog()
#在提取类成员函数的时候，通常都是利用类方法的手段
Dog.age=23#直接能够修改类内的成员
print(Dog.age)#23可以发现，成员我无论从变量访问还是成员访问，都是访问的到的
#print(a.work())
print(Dog.work())#通过类无法访问函数,从而引出静态方法
